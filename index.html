Tsunami Projekt: User Documentation

Installing and Running

  1. clone the repository with git clone https://github.com/darafsa/tsunami_lab.git

  2. add and update the submodules with git submodule init and git submodule update

  3. build with scons

  4. run the solver with ./build/tsunami_lab CELLS SOLVER SETUP BOUNDARYLEFT BOUNDARYRIGHT [height] [velocity] [endTime]

  5. execute the tests with ./build/tests

Command line parameters when executing

CELLS = Number of cells in x and y direction (where number >= 1)
SOLVER = Type of the solver which (ROE or FWAVE)
SETUP = Setup to use (DAMBREAK, RARE, SHOCK, BATHYMETRY, SHOCKREFLECT, DAMBREAK2D or BATHYMETRY2D)
BOUNDARY{LEFT/RIGT} = Boundary condition to use (OUTFLOW, REFLECTING]
[height, velocity] (optional) = The height and velocity to use for RareRare and ShockShock Setup
[endTime] (optional) = The time the simulation runs
